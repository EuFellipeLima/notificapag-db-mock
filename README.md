# 🗄️ NotificaPag - Database & Mock

Repositório destinado a armazenar os scripts de banco de dados, arquivos JSON estruturados e dados falsos (mockados) utilizados para testar o fluxo de informações do ecossistema NotificaPag em ambiente de desenvolvimento e apresentações acadêmicas.

## 🚀 Estrutura
* Estruturas JSON de Clientes e Parcelas.
* Scripts de simulação de banco de dados.
